# Format `lemmas`

CSV:
- word:  word lemma in the respective language.
- lang:  language code.
- valid: whether the lemma is valid, used for filtering / designating the
  subset of used lemmas.
- info:  additional info (e.g., etymology).

# Format `conloan_annotation`

XLSX[old]:
- Loanword sentence
- Native sentence
- Target
- Valid
- Donor lang
- Donor word
- Suggestions
- Etymology

XLSX[new and simplified]:
- Loanword
- Native
- Target
- Suggestions
- Info 

# Format `conloan_annotation`

CSV[standard]:
_defined upstream_

CSV[context switch extension]:
CSV[standard] + <CS1>...</CS1> | <CN1>...</CN2> annotations for context-switches.
